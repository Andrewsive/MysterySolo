# 🏢 回声公寓 - 剧本优化说明

## 🔧 问题诊断

### 用户反馈
> "点击选择的行动后没有反应，需要设置后续相应的不同反馈"

### 根本原因
"回声公寓"剧本的场景文字**过于简短**，缺乏：
1. ❌ 明确的任务目标说明
2. ❌ 选项的详细标注（为什么要选）
3. ❌ 线索获得的反馈提示
4. ❌ 推理过程的引导
5. ❌ 玩家行动后的清晰反馈

**对比示例：**

优化前（模糊）：
```
夜深的公寓大堂只剩值班灯。回声在楼梯间回荡。

选项：
· 上楼寻找声源
· 敲响管理员室
```

优化后（清晰）：
```
【深夜·公寓大堂】

时钟指向凌晨 1 点。你站在转手多次的公寓大堂里...

🎯 【任务目标】
· 调查公寓，寻找回声的来源
· 询问管理员或住户了解情况
· 解开这栋公寓的秘密

选项：
· 🔺 上楼寻找声源（直接调查）
· 🚪 敲响管理员室（询问线索）
```

## ✨ 优化内容

### 1. 开场场景强化

**新增内容：**
- 🎯 明确的任务目标列表
- 💡 新手提示
- 🕐 时间背景（凌晨1点）
- 🏢 场景氛围描写

### 2. 选项标注系统

**所有选项添加：**
- 📋 emoji 图标（🔺 🚪 📋 📽️等）
- 括号说明（直接调查、询问线索、深入分析）
- 后果提示（进入关键场景、收集更多线索）
- 结局标注（→ 真结局、→ BE结局）

### 3. 线索反馈强化

**线索获得时显示：**
```
✅ 【获得线索 1/2】《栀子花香》
💭 推理线索：为什么会有栀子花的香味？这和回声有关吗？
```

**特点：**
- 显示进度（1/2）
- 给出推理问题
- 引导思考方向

### 4. 推理引导场景

**新增"档案分析"场景：**
```
【档案分析】

📋 所有搬走的住户都在夜里听见同样的脚步声
📋 脚步声有固定的节奏："哒...哒哒...哒..."
📋 档案里有一张泛黄的照片：一位年轻的舞者
📋 照片背面写着："301 室，每晚 1 点"

💡 【推理串联】
   · 栀子花香 + 舞者照片
   · 固定的脚步节奏 = 舞步？
   · 301 室...就在二层储物间旁边

现在明白了：回声不是灵异现象，而是某种"残留"...
```

**作用：**
- 整理已收集的线索
- 引导玩家推理
- 揭示线索之间的联系

### 5. 场景分支优化

**优化前（线性）：**
```
lobby → stairs → storage（结局）
lobby → concierge → files → storage（结局）
```

**优化后（网状）：**
```
lobby → stairs → storage（线索不足警告）
      ↓           ↓
    lobby_return  ending_echo_incomplete（可返回）
      ↓
    concierge → files → storage_prepared（完整真相）
                         ↓
                      ending_echo（真结局）
```

**改进点：**
- 增加返回选项（lobby_return, stairs_informed）
- 区分收集线索的完整度（storage vs storage_prepared）
- 提供重新调查的机会（ending_echo_incomplete）

## 🎮 游戏流程示例

### 完美通关路线（收集所有线索）

```
1. 【大堂】选择 "🚪 敲响管理员室"
   → 获得《搬家档案》线索 (1/2)

2. 【管理员室】选择 "📋 仔细查看档案细节"
   → 进入档案分析，看到推理串联

3. 【档案分析】自动引导
   → 了解舞者、栀子花、脚步节奏的联系

4. 【档案分析】选择 "🔺 前往二层储物间"
   → 顺路获得《栀子花香》线索 (2/2)

5. 【储物间·真相时刻】选择 "📽️ 打开投影仪"
   → 真结局（85分）
```

### 快速通关路线（跳过线索）

```
1. 【大堂】选择 "🔺 上楼寻找声源"
   → 获得《栀子花香》线索 (1/2)

2. 【二层走廊】选择 "🚪 推开储物间"
   → 进入储物间，但缺少关键线索

3. 【储物间·未知真相】系统警告
   → "⚠️ 但你没有收集所有线索，可能会错过重要信息..."

4. 选择：
   · 打开投影仪 → 不完整的真相（可返回补线索）
   · 取走纸箱 → BE结局（55分）
```

## 💡 设计理念

### 1. 温暖的引导而非冰冷的惩罚

**不完整通关不会直接BE：**
- 会给予警告："可能会错过重要信息"
- 提供返回补充线索的机会
- 即使跳过线索也能看到基本真相

### 2. 鼓励探索而非强制收集

**两种玩法都支持：**
- 快速通关：15分钟，能看到基本结局
- 完美通关：20分钟，解锁完整故事

### 3. 情感共鸣优先于解谜难度

**剧情主题：**
- 舞者的遗憾
- 回声的温柔
- 记忆的传承

**结局设计：**
- 真结局：将投影仪送回家属，让回声归于平静
- BE结局：成为下一个见证者，回声继续轮回
