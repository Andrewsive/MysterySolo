## 🕹️ 如何游玩

### 1. 阅读剧情文本
- 每个场景都会有**详细的环境描述**
- 注意**标题**：【书房环境】、【线索：手札】等
- 留意特殊标记：
  - `💡` = 提示
  - `💭` = 推理线索
  - `⏰` = 时间进度
  - `✅` = 获得线索
  - `🎯` = 任务目标

### 2. 查看可选行动
- 场景下方会显示 **2-3 个选项按钮**
- 每个选项都有**图标**和**说明**：
  ```
  📖 翻阅手札（看起来很重要）
  🔑 检查钥匙（上面有刻字）
  ```
- **括号内的文字**会告诉你这个选项的作用

### 3. 做出选择
- 点击任意选项按钮，剧情会继续
- 某些选项会**获得线索**（顶部进度条会增加）
- 某些选项会**推进剧情**（进入新场景）
- **不用担心选错**，游戏鼓励你大胆探索！

### 4. 收集线索
- 探索过程中会获得线索，如：
  ```
  ✅ 【获得线索 1/4】《神秘手札》
  ```
- 顶部的**线索进度条**会实时更新
- 点击 **📋 线索簿** 按钮可以查看已收集的线索

### 5. 使用辅助功能
- **📋 线索簿**：查看所有已收集的线索和详细描述
- **💡 提示**：如果卡关，可以查看提示（会影响最终评分）
- **🔄 重新开始**：重置游戏进度，从头开始

## 💡 游戏技巧

### 🔍 探索阶段（收集线索）
1. **不要急于推进剧情**
   - 尽量收集完所有线索再前进
   - 选项中标注"还有其他线索"的，建议先探索
   
2. **仔细阅读每条线索**
   - 线索中的**数字、日期、地点**很重要
   - 多条线索会互相印证

3. **善用线索簿**
   - 收集 2 条以上线索后，会出现"推理建议"
   - 定期查看线索簿，整理思路

### 💭 推理阶段（串联线索）
1. **寻找共同点**
   - 不同线索中重复出现的信息是关键
   - 例如："1201"在多处出现，说明很重要

2. **注意场景中的推理提示**
   - `💭 推理线索：` 会给出思考方向
   - 专门的"推理时刻"场景会帮你总结线索

3. **时间线很重要**
   - 注意剧情中的时间进度
   - 某些线索会提到具体日期

### 🚪 决策阶段（选择结局）
1. **关键决策会有明确标注**
   ```
   🔓 用钥匙打开边门（勇敢直面过去）→ 真结局
   😰 太可怕了，逃离这里（放弃挑战）→ BE结局
   ```

2. **不确定时可以"重新思考"**
   - 某些选项允许你返回重新分析线索
   - 不会浪费进度，放心尝试

3. **勇敢做出选择**
   - 推理游戏的乐趣在于体验不同结局
   - 可以重新开始，尝试不同路线

## ❓ 常见问题

### Q: 我不知道该点哪个选项？
**A:** 不用担心！每个选项都有说明：
- 括号内的文字会告诉你这个选项的作用
- 标注"还有其他线索"的选项建议先探索
- 标注"推进剧情"的选项会进入下一阶段

### Q: 我会错过重要线索吗？
**A:** 游戏设计上会引导你收集：
- 顶部有线索进度显示（如 2/4）
- 选项会提示"可能遗漏重要信息"
- 推理时刻场景会总结已收集的线索

### Q: 我卡关了怎么办？
**A:** 有多种帮助方式：
1. 点击 **💡 提示** 按钮查看提示（最多2条）
2. 打开 **📋 线索簿**，查看已收集的线索
3. 选择"重新思考"选项，系统会再次分析线索
4. 重新开始游戏，尝试不同的探索顺序

### Q: 提示会影响游戏吗？
**A:** 会影响最终评分：
- 无提示通关：满分
- 使用1次提示：扣少量分
- 使用2次提示：扣更多分
- 但不会影响结局，放心使用！

### Q: 有多少个结局？
**A:**《午夜书房》有 2 个结局：
- **真结局**（92分）：勇敢打开边门，直面过去
- **BE结局**（32分）：逃避挑战，困于书房
